# cli_app_scaffold

Scaffold a new CLI app with ink + React in seconds.

## Usage

```bash
node src/index.js my-app
cd my-app
npm install
npm run dev
```

## What You Get

- Fullscreen terminal app with alternate screen buffer
- State engine with navigation history (back button works)
- Theme system with colors
- Reusable components (Cursor, HelpFooter, Separator)
- Home screen with menu
- Example screen to get started
- TypeScript + tsx for development

## Stack

- React 19 + ink 6 (React for terminal)
- TypeScript 6
- tsx for dev, tsc for build
